odrix-cli
=========

A simple CLI for ODRIX (ODIN@matrix)

# Usage
<!-- usage -->
```sh-session
$ git clone 
$ npm install  odrix-cli
$ odrix-cli COMMAND
running command...
$ odrix-cli (-v|--version|version)
odrix-cli/0.0.0 darwin-x64 node-v14.18.1
$ odrix-cli --help [COMMAND]
USAGE
  $ odrix-cli COMMAND
...
```
